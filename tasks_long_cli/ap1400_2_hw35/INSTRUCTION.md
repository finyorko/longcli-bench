This task requires you to complete both AP1400_2 HW3 and HW5 assignments simultaneously in the ap1400_2_hw folder.

Please read the code and corresponding README.md files in the ap1400_2_hw directory to understand the requirements for both assignments.

## HW3 Requirements
You need to implement a Binary Search Tree (BST) data structure as specified in the HW3 README.md. This includes implementing the Node class and the BST class. The test suite includes 31 tests covering all aspects of the BST implementation.


## HW5 Requirements  
You need to implement a coffee shop system with ingredients and espresso-based drinks (Cappuccino and Mocha) as specified in the HW5 README.md.

Both assignments must be completed correctly for the tests to pass. The test suite includes 31 tests for HW3 and 12 tests for HW5, for a total of 43 tests.
